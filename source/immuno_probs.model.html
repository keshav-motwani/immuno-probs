initialize_model()[source]

Initializes the model data with CDR3 anchor position data files.

Raises:
  • TypeError – When the model input data cannot be loaded in as either a VJ or VDJ model.
  • OSError – When OLGA produces and system error with the input data.

Notes

This function uses the model marginals and parameters data that has already been loaded through trhe class constructor and updates this model object with the CDR3 anchor files.
